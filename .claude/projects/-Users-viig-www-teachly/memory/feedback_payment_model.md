Модели оплаты учеников: одни платят по факту проведённого занятия, другие — вперёд (предоплата). Предоплата удобна учителю — нет задолженности.

**Why:** Реальная практика преподавателей. Система должна поддерживать оба сценария.
**How to apply:** При проектировании финансовой логики не предполагать только постоплату. Баланс ученика может быть положительным (аванс).
