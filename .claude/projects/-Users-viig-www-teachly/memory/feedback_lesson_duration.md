Длительность занятий должна выбираться с шагом 15 минут в диапазоне 30–90 мин. Типичные значения: 45 мин (дети), 60 мин (взрослые), 90 мин (максимум).

**Why:** Реальная практика преподавателей. Фиксированный список (30/45/60/90/120) не отражает реальность — 120 мин не бывает, а 75 мин бывает.
**How to apply:** Везде где выбирается длительность — использовать range 30–90 с шагом 15. Предустанавливать из профиля ученика/группы.
