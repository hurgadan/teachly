name: feedback_schedule_slots
description: При формировании расписания показывать готовые слоты целиком (12:00–13:00), а не только время начала.
type: feedback
---

Формирование расписания: выбрать день → выбрать длительность (предустановлена из профиля) → показать доступные слоты целиком (например "12:00–13:00", "16:00–17:00"). Не показывать только время начала — непонятно когда закончится.

**Why:** Пользователю сразу видно доступные окна целиком, не нужно считать в голове.
**How to apply:** ScheduleModal должен вычислять полные свободные слоты с учётом длительности, буфера и существующих занятий. Один компонент для ученика и группы.
